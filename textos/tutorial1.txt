Tutorial 1

¿Como instalar las librerias de JFace?

1.- Primero descargue el paquete SWT: The Standard Widget Toolkit desde http://www.eclipse.org/swt/ 

- Pulse el botón derecho sobre el proyecto y seleccione Build Path > Configure Build Path
Seleccione la pestaña Libraries y pulse el botón Add Library.
Seleccione la librería SWT pulse Next y OK para finalizar añadiendo las librerías SWT a su proyecto.

org.eclipse.swt_<version info>.jar

2.- Segundo, añada tambien estas librerias:

org.eclipse.core.commands_<version info>.jar
org.eclipse.equinox.common_<version info>.jar
org.eclipse.jface_<version info>.jar
org.eclipse.osgi_<version info>.jar
org.eclipse.ui.workbench_<version info>.jar
